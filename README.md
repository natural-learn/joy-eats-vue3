# 悦享外卖商家管理后台

## 项目介绍
一款面向中小外卖商家的一站式运营管理后台，覆盖订单处理、菜品/套餐管理、数据统计、员工管理等核心场景，采用前后端分离架构开发，旨在解决外卖商家日常运营效率低、数据管理混乱的问题。

## 技术栈
### 前端
- 框架：Vue 3 (Composition API)
- 构建工具：Vite
- UI组件库：Element Plus
- 网络请求：Axios
- 状态管理：Pinia

## 项目运行步骤
### 前端

# 克隆项目
git clone https://github.com/natural-learn/joy-eats-vue3.git

# 安装依赖
npm install

# 启动开发服务
npm run dev
